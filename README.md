# 🧠 Memories – 기억력 강화 웹 애플리케이션
> 로그인 기반 CRUD 서비스 구현을 목표로 한 **개인 풀스택 프로젝트**

## 📌 프로젝트 한 줄 요약
사용자 인증을 기반으로 개인 학습 데이터를 관리하는 기억력 강화 웹 서비스

## 🎯 프로젝트 목적
- 로그인 기반 서비스 구조 이해
- Spring Boot + React 풀스택 개발 경험
- CRUD 중심 애플리케이션 설계 역량 강화

## 🧩 핵심 구현 기능
### 🔐 인증
- 회원가입 / 로그인
- 사용자별 데이터 접근 제어

### 📝 기억 데이터 관리
- 기억 데이터 CRUD
- 사용자별 데이터 분리 저장
- 카테고리 기반 분류 및 정렬

## 🛠 기술 스택
### Backend
- Java 21
- Spring Boot 3.x
- Spring Security
- JPA / Hibernate
- MySQL 8.0

### Frontend
- React
- TypeScript
- Vite
- Axios

## 🏗 설계 포인트
- 인증과 비즈니스 로직 분리
- Controller / Service / Repository 계층 분리
- 사용자 중심 데이터 모델링

## 💡 배운 점
- 로그인 기반 서비스의 인증 흐름 이해
- CRUD 애플리케이션의 구조적 설계 경험
- 프론트엔드와 백엔드 연동 경험
